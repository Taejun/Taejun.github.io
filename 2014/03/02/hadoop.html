Hadoop src build (x64)

02 Mar 2014

** 기존 hadoop-2.2.0 의 경우 x86 binary만 존재 하므로, 64bit binary를 얻기위해 MVN 이용한 hadoop build 절차 문서 입니다.

1) Requirment

  • JDK 1.7_x64 ( Oracle web site )
  • Maven (http://maven.apache.org/download.cgi )
  • protobuf (https://code.google.com/p/protobuf/downloads/detail?name=protobuf-2.5.0.tar.bz2 )
  • Development tool ( gcc , cmake , autoconf , zlib-dev libssl-dev, libtool etc… )
  • Openssl-dev , ant

2) Hadoop Source download

~]# svn co https://svn.apache.org/repos/asf/hadoop/common/tags/release-2.2.0

3) Maven install

wget http://apache.tt.co.kr/maven/maven-3/3.1.1/binaries/apache-maven-3.1.1-bin.tar.gz 
tar xvzf apache-maven-3.1.1-bin.tar.gz 
mv apache-maven-3.1.1 /usr/local/maven
export MVN_HOME=/usr/local/maven
export PATH=$PATH:$MVN_HOME/bin

4) JDK install (default)

5) yum install openssl or apt-get install openssl

6) protobuf (https://code.google.com/p/protobuf/downloads/detail?name=protobuf-2.5.0.tar.bz2 )

wget https://protobuf.googlecode.com/files/protobuf-2.5.0.tar.bz2 
tar xvzf protobuf-2.5.0.tar.bz2
./configure ; make ; make install
ldconfig

7) ant install (http://ant.apache.org/bindownload.cgi ) export path…

8) Hadoop compile

cd  Hadoop-src
mvn compile –X
mvn package –Pdist,native –DskipTests –Dtar
