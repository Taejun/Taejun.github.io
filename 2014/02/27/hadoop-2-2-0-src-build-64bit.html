hadoop 2.2.0 src build 64bit

27 Feb 2014
HADOOP SRC build
  1. Requirment
-       JDK 1.7_x64 ( Oracle web site ) -       Maven (http://maven.apache.org/download.cgi  ) -       protobuf (https://code.google.com/p/protobuf/downloads/detail?name=protobuf-2.5.0.tar.bz2  ) -       Development tool ( gcc , cmake , autoconf , zlib-dev libssl-dev, libtool  etc… ) -        Openssl-dev , ant
  1. Hadoop Source download
~]# svn co https://svn.apache.org/repos/asf/hadoop/common/tags/release-2.2.0  
  1. Maven install
-       ~]# wget http://apache.tt.co.kr/maven/maven-3/3.1.1/binaries/apache-maven-3.1.1-bin.tar.gz -       tar xvzf apache-maven-3.1.1-bin.tar.gz -       mv apache-maven-3.1.1 /usr/local/maven -       export MVN_HOME=/usr/local/maven -       export PATH=$PATH:$MVN_HOME/bin  
  1. JDK install  (default)
  2. yum install openssl  or apt-get install openssl
  3. protobuf (https://code.google.com/p/protobuf/downloads/detail?name=protobuf-2.5.0.tar.bz2 )
-       wget https://protobuf.googlecode.com/files/protobuf-2.5.0.tar.bz2 -       tar xvzf protobuf-2.5.0.tar.bz2 -       ./configure ; make ; make install -       ldconfig    
  1. ant install (http://ant.apache.org/bindownload.cgi )

export path…

  1. Hadoop compile
-       Cd  Hadoop-src -       Mvn compile –X -       Mvn package –Pdist,native –DskipTests –Dtar       Link http://pravinchavan.wordpress.com/2013/04/14/building-apache-hadoop-from-source/ http://www.micmiu.com/opensource/hadoop/hadoop-build-source-2-2-0/ http://jugnu-life.blogspot.com/2013/09/build-and-compile-hadoop-from-source.html https://issues.apache.org/jira/browse/HADOOP-10117 https://svn.apache.org/repos/asf/hadoop/common/trunk/BUILDING.txt http://wiki.apache.org/hadoop/HowToContribute http://blog.csdn.net/linshao_andylin/article/details/12307747
